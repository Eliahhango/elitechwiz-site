# EliTechWiz Website

Official static website for [EliTechWiz](https://elitechwiz.site) — a UI/UX design, branding, and web development agency.

## 🌐 Live Site

[https://elitechwiz.site](https://elitechwiz.site)

## 📁 Project Structure

This site was built and exported from [Webflow](https://webflow.com). It is hosted as a static site.

| Folder / File | Description |
|---|---|
| `index.html` | Homepage |
| `about/` | About page |
| `blog/` | Blog listing page |
| `contact/` | Contact page |
| `pricing/` | Pricing page |
| `services/` | Service sub-pages (brand identity, UX audit, logo design, etc.) |
| `solutions/` | Solution sub-pages (MVP, etc.) |
| `industries/` | Industry sub-pages (web3, etc.) |
| `works/` | Portfolio / works page |
| `referral/` | Referral program page |
| `cookie-policy/` | Cookie policy |
| `privacy-policy/` | Privacy policy |
| `editorial-policy/` | Editorial policy |
| `assets/` | Static assets (SVG logos, images) |

## 🚀 Deployment

This is a static HTML site. Deploy by uploading the contents to any static host (GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.).
